## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Juninhoww2/Nextjs-crud-tester
```

Entre no diretório do projeto

```bash
  cd Nextjs-crud-tester
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
